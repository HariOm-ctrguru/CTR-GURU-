# CTR GURU

CTR Guru is a web-based thumbnail CTR analyzer for YouTube creators.

It estimates click-through rate (CTR) using image properties like brightness,
contrast, and niche patterns, and provides actionable suggestions to improve
thumbnail performance.

This is an MVP built using HTML, CSS, and JavaScript.

---

## Features
- Thumbnail upload & preview
- Estimated CTR score
- Improvement suggestions with CTR impact
- Niche-based analysis
- Free + Pro access (manual unlock)

---

## How It Works
CTR Guru analyzes visual factors such as:
- Brightness
- Contrast
- Subject clarity
- Niche weighting

⚠️ CTR values are estimates, not YouTube internal data.
